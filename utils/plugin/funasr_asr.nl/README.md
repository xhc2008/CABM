# 插件/扩展

本插件是示范插件，示范如何修改内容，重写前端函数，创建后端路由

## 插件/扩展使用限制

本插件仅限有独立显卡且显卡显存>=2GB的用户使用

## 插件/扩展介绍

本插件是示范插件，示范如何修改内容，重写前端函数，创建后端路由

本插件修改了原先的语音识别换成了阿里巴巴达摩院的funasr，可以识别出emoji，对于中文识别能力更强，速度稍慢一些

## 插件/扩展使用

首先在你的环境安装本插件的requirements.txt

然后将插件目录的.nl删除掉(nl=not load也就是不加载这个插件)
